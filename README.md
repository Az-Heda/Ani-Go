# Ani-Go \[BETA\]

Webserver in Golang for a website dedicated to Anime

### Compile:
```sh
>>> go build -c AniGo.exe Ani.go
>>> ./AniGo.exe
```

### Run witout compiling:
```sh
go run Ani.go
```
The server will run on **http://127.0.0.1:8770**
